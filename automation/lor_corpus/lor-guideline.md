# LOR structure

Transcribed from `lor-guideline.pdf` — handwritten 8-point structure note from the
operator. This is the canonical structure every drafted Letter of Recommendation
must follow. The runbook agent reads this file before composing any LOR.

1. **Teacher introduces herself** — time period of association, the subject she
   teaches, and recognises the student's characteristics.

2. **Student's academic characteristics** — state some key topics of grade XII in
   which the student excels.

3. **Student's classroom participation** — concept clarity etc.

4. **Student's class project** — her planning of the project, presentation,
   objective of the project and how it was fulfilled. Some bit of praise.

5. **Mention research paper.**

6. **Mention an academic activity** in which the student excelled.

7. **Talk of leadership** and her position with peers.

8. **Finally round off.**

---

The handwritten `lor-guideline.pdf` is the authoritative source. This markdown
mirror exists so the runbook agent can read the structure as text without OCR.
If the operator updates the handwritten note, re-transcribe here.
